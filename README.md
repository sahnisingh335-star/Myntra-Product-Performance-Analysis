# Myntra Product Performance Analysis

## Project Overview

This project presents an interactive Power BI dashboard developed to analyze Myntra product data and generate meaningful business insights.

## Key Features

* Product Performance Analysis
* Revenue Analysis
* Brand Contribution Analysis
* Discount Insights
* Customer Rating Analysis
* Product Segmentation
* Statistical Analysis
* Interactive Dashboard Visualizations

## Tools & Technologies

* Power BI
* Power Query
* Python
* Pandas
* NumPy
* Matplotlib

## Files Included

* Data Cleaning Scripts
* Brand Performance Analysis
* Statistical Analysis
* Product Segmentation
* Recommendation Engine
* Product Success Prediction
* Dashboard Assets

## Business Insights

The dashboard helps identify:

* Top-performing brands
* Revenue contribution by brand
* Product popularity trends
* Discount distribution patterns
* Customer rating insights
